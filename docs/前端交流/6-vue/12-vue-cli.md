## vue-cli

?> 脚手架是通过webpack搭建的开发环境---基于node

- 使用ES6方法。
- 打包压缩js为一个文件
- 项目文件在环境中编译，而不是浏览器
- 实现页面的自动刷新

### 安装node  

### 安装vue-cli    

安装命令：

    npm install vue-cli -g

3.0以上版本：

    npm install -g vue@cli

### 创建项目文件夹

创建项目命令

    vue init webpack 项目名

3.0以上版本

    vue create 项目名

运行项目 

    cd到项目文件夹

运行命令  

    npm run dev

vue-cli 3.0以上  

    npm run serve

### 文件结构的介绍

安装路由的vuex  

    npm install vuex

安装axios    

    npm install axios

安装vue-resource

    npm install vue-resource
