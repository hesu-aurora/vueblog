**跨端兼容**

条件编译是用特殊的注释作为标记，在编译时根据这些特殊的注释，将注释里面的代码编译到不同平台。

写法：

以 `#ifdef` 或 `#ifndef` 加 `%PLATFORM%` 开头，以 `#endif` 结尾。

    #ifdef：if defined 仅在某平台存在
    #ifndef：if not defined 除了某平台均存在
    %PLATFORM%：平台名称

| 条件编译写法 |	说明 |
|:---|:---|
| `#ifdef APP-PLUS`<br/> 需条件编译的代码</br>`#endif` | 仅出现在 App 平台下的代码 |
| `#ifndef H5`<br/> 需条件编译的代码<br/> `#endif` | 除了 H5 平台，其它平台均存在的代码 |

> `%PLATFORM%` 可取值如下：

    APP-PLUS	App
    APP-PLUS-NVUE	App nvue
    H5	H5
    MP-WEIXIN	微信小程序
    MP-ALIPAY	支付宝小程序
    MP-BAIDU	百度小程序
    MP-TOUTIAO	头条小程序
    MP-QQ	QQ小程序
    P	微信小程序/支付宝小程序/百度小程序/头条小程序/QQ小程序

> 支持的文件

    .vue
    .js
    .css
    pages.json
    各预编译语言文件，如：.scss、.less、.stylus、.ts、.pug

js使用 

    // 注释
    
css 使用 

    /* 注释 */
    
vue/nvue 模板里使用 

    <!-- 注释 -->

API 的条件编译

    // #ifdef  %PLATFORM%
    平台特有的API实现
    // #endif

**static 目录的条件编译**

`static` 目录下新建不同平台的专有目录（目录名称同 `%PLATFORM%` 值域,但字母均为小写），专有目录下的静态资源只有在特定平台才会编译进去。

    ┌─static                
    │  ├─mp-weixin
    │  │  └─a.png     
    │  └─b.png
    ├─main.js        
    ├─App.vue      
    ├─manifest.json 
    └─pages.json
